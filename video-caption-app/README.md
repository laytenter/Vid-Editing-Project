# Video Caption App

Electron + TypeScript desktop app for generating captions and clipping video segments locally using FFmpeg and Whisper.  
Runs entirely offline with no cloud processing.

**Technologies:** Electron, TypeScript, FFmpeg, Whisper, local AI transcription

**Status:** Early development / experimental

## Why this project exists

This project provides a privacy-friendly, local workflow for generating captions and clipping segments from videos without uploading media to cloud services.

## Features

- Select video files via file dialog or drag and drop
- Extract mono 16 kHz WAV audio with FFmpeg
- Generate captions locally with Whisper runtime binaries
- Save captions as `.srt` or `.vtt`
- Configure and persist output directory settings
- View live tool logs with filtering controls
- Create video clips using start/end timestamps and FFmpeg mode selection

## Project structure

- `src/` - TypeScript and static source files (`main`, `preload`, `renderer`, HTML/CSS)
- `dist/` - compiled output generated by `npm run build`
- `bin/` - local runtime binaries (FFmpeg + Whisper runtime files)
- `scripts/` - helper scripts (for example static file copy and Windows setup)

## Installation

### Requirements

- Node.js 18+ (recommended: current LTS)
- Windows environment for bundled runtime binaries

### Setup

```powershell
npm install
npm run build
npm start
```

## Runtime dependencies

This repository currently includes FFmpeg and Whisper runtime binaries in `bin/`.

Expected runtime files include:

- `bin/ffmpeg.exe`
- `bin/ffprobe.exe`
- `bin/whisper.exe`

Whisper model files are loaded from the user models directory (for example `%APPDATA%\video-caption-app\models\`).  
Models must currently be placed there manually by the user. Automatic model download may be added in a future release.

## Example workflow

Typical workflow:

1. Load a recorded interview or podcast video.
2. Extract audio and generate captions locally.
3. Review the generated `.srt` file.
4. Identify interesting segments.
5. Enter timestamps and create short video clips.

## Basic usage

1. Select a video file (button or drag/drop).
2. Click **Extract Audio**.
3. Click **Generate Captions**.
4. Export captions with **Save SRT** or **Save VTT**.
5. Create clips by entering start/end timestamps, selecting mode, and clicking **Create clip**.

## Current limitations

- Currently Windows-focused (bundled FFmpeg and Whisper binaries are Windows builds).
- Whisper models are not automatically downloaded yet.
- Clip creation currently requires manual timestamp input.
